# SerialPortScanner Junsun

Proiect Android care scanează porturile UART (/dev/tty*) pe dispozitive AllWinner / Junsun 8227L pentru a detecta comunicarea CAN (CANBox Hiworld PA12.xx).

## Ce contine
- Scanare automată a porturilor obișnuite
- Wrapper SerialPort (NDK) pentru a deschide /dev/tty* fără root (dacă permisiunile permit)
- C/C++ code + CMake pentru a construi libserial_port.so

## Build
1. Deschide în Android Studio sau folosește CLI Gradle.
2. Dacă folosești Android Studio, Gradle va compila și .so automat (vezi `externalNativeBuild`).
3. Dacă ai deja `libserial_port.so`, pune-le în `app/src/main/jniLibs/armeabi-v7a/` și `app/src/main/jniLibs/arm64-v8a/`.

## Observatii
- Pe multe unități Junsun, portul CanBox este `/dev/ttyS3` la 9600 sau `/dev/ttyS1` la 38400.
- Dacă aplicația nu reușește să deschidă un port din cauza permisiunilor, ai două opțiuni: rulezi cu firmware care expune porturile, sau copiezi .so dintr-un repo compatibil (unele binare vin cu permisiuni diferențiate).
